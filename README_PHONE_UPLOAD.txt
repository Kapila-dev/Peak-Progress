Peak Progress PWA v25 - GitHub Pages Ready

Phone upload steps:
1. Go to github.com in Chrome and sign in.
2. Create a new repository named peak-progress. Keep it Public for easiest GitHub Pages setup.
3. Open the repository, tap Add file, then Upload files.
4. Upload everything inside this folder: index.html, manifest.webmanifest, sw.js, .nojekyll, and the icons folder/files.
5. Commit the files.
6. Go to repository Settings, then Pages.
7. Source: Deploy from a branch. Branch: main. Folder: /root. Save.
8. Wait a minute or two, then open the Pages URL.
9. In Chrome on Android, tap the three-dot menu and choose Add to Home screen or Install app.

Important:
- Do not upload the zip itself to GitHub Pages. Upload the files inside the extracted folder.
- Data is stored locally on the phone/browser. Use Data Vault export inside the app for backups.
- If an old build appears, close the installed app and reopen the Pages URL in Chrome once.
